Le Robot

Ceci est la page principale de description des tests d'acceptations sur le robot.

Caractéristiques du robot

Contrôle et observations

Contrôles

  • Faire atterrir le robot aux coordonnées X, Y. (Cette commande est inopérante lorsque le robot est au sol)
  • Déplacer le robot :
    • Aller en avant
    • Aller en arrière
    • Tourner à droite
    • Tourner à gauche
  • Faire une cartographie autour du robot
  • Déployer les capteurs solaires pour un temps donné

Observations

  • La carte du terrain découvert
  • Le niveau d'énergie courant
  • la position et l'orientation courante

Gestions spécifiques

Le robot possède certaines spécificités :

  • Pour la gestion de la cartographie
  • Pour la gestion des déplacements
  • Pour la gestion de l'énergie

État initial

Au départ, le robot se trouve en vol, il atterrit à des coordonnées qui lui sont spécifiées, son orientation initiale est vers le nord. Ces coordonnées correspondent ensuite à son point de départ pour toute action qu'il souhaite effectuer.

La batterie est chargée à 20 unités.
